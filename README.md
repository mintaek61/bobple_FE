<h2>✍🏻 프로젝트 개요</h2>
바쁜 일상 속에서 매번 식사 메뉴를 결정하는 데 어려움을 겪고 있는 사용자들을 위한 
메뉴 추천 서비스와, 다양한 게임 요소와 커뮤니티 기능을 접목한 플랫폼을 제공합니다
